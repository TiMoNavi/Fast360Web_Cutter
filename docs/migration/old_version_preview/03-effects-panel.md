# 特效面板 (PcEffectsPanel)

## 位置
右侧，在工作台面板下方或独立区域

## 组件结构

### 3.1 特效分类

#### Transition（过渡）
- **Black fade** - 黑色淡入淡出（900ms）
- **White fade** - 白色淡入淡出（720ms）
- **Flash cut** - 闪光切换（260ms）
- **Neon wipe** - 霓虹擦除（900ms，青色）
- **Grid dissolve** - 网格溶解（900ms，模糊）
- **VHS blank** - VHS空白（700ms）

#### Color（调色）
- **Cyan boost** - 青色增强
- **Magenta wash** - 洋红渲染
- **Sunset grade** - 日落调色
- **Cold chrome** - 冷色金属
- **Warm VHS** - 暖色VHS
- **Mono scan** - 单色扫描

### 3.2 特效操作

#### 添加特效
- **点击特效项** - 在当前时间点添加特效
- **预览功能** - 鼠标悬停时预览特效效果

#### 特效参数
- **持续时间** - 某些特效有预设持续时间
- **特效参数** - 如颜色、强度等
