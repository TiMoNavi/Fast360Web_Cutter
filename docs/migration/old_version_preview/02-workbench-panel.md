# 工作台面板 (PcWorkbenchPanel)

## 位置
右侧，垂直布局

## 组件结构

### 2.1 面板头部
- **窗口装饰：** 三个彩色圆点（洋红、青、橙）
- **标题：** `EDITOR_WORKBENCH`
- **折叠按钮：** 可以折叠/展开面板

### 2.2 视口控制区域

#### FOV控制
- **当前FOV显示：** `FOV: 90°`
- **FOV减小按钮：** `FOV -` 或 `-`
- **FOV增加按钮：** `FOV +` 或 `+`

#### 视口移动控制
- **Pitch Up按钮：** `↑` 向上移动视口
- **Pitch Down按钮：** `↓` 向下移动视口
- **Yaw Left按钮：** `←` 向左移动视口
- **Yaw Right按钮：** `→` 向右移动视口

#### 锁定控制
- **锁定按钮：** `🔒 Lock` / `🔓 Unlock`
- **状态显示：** 显示当前是否锁定

### 2.3 遮罩控制区域

#### 遮罩透明度
- **透明度滑块：** 0.0 - 1.0
- **当前值显示：** 例如 `0.7`

### 2.4 时间轴控制区域

#### 录制控制
- **开始录制按钮：** `Start Crop`
- **结束录制按钮：** `End Crop`
- **录制状态显示：** `idle` / `recording` / `ready` 等
- **状态消息：** 显示当前操作的详细信息

#### 时间轴操作
- **刷新按钮：** `Flush` - 刷新路径到后端
- **切点按钮：** `Cut Here` - 在当前位置标记切点

### 2.5 渲染控制区域

#### 渲染按钮
- **渲染按钮：** `Render` - 触发后端渲染
- **渲染状态：** 显示渲染进度

#### 自动渲染开关
- **复选框：** `Auto Render`
- **说明：** 录制结束后自动触发渲染

#### 下载区域
- **下载链接：** 渲染完成后显示
- **导出ID显示：** 显示导出文件的ID

### 2.6 Timeline状态显示

#### 路径状态
- **最后接受的patch：** 显示revision号
- **接受的点数：** 显示有多少个路径点被接受
- **待处理的点：** 显示缓冲中的点数
- **队列中的批次：** 显示等待发送的批次数

#### 时间范围
- **第一个点时间：** 显示路径起始时间
- **最后一个点时间：** 显示路径结束时间
