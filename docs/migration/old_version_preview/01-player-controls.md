# 旧版本UI完整清单

## 1. 播放控制条 (PcPlayerControls)

### 位置
底部，横跨整个屏幕

### 组件结构

#### 1.1 状态条
- **位置：** 播放控制条上方
- **内容：**
  - 提示符：`>`
  - 文本：`WEBXR SESSION`
  - 播放状态：`playing` / `paused` / `loading` 等
  - 视频格式：`MP4` / `HLS`

#### 1.2 窗口装饰
- **三个彩色圆点：**
  - 洋红色圆点
  - 青色圆点
  - 橙色圆点
- **标签：** `PLAYBACK_CORE // 2088`

#### 1.3 进度条区域
- **当前时间显示：** `MM:SS` 格式
- **进度条滑块：** 可拖动，显示播放进度
- **总时长显示：** `MM:SS` 格式

#### 1.4 主控制按钮
- **上一个视频按钮：** `⏮` 图标
- **播放/暂停按钮：** `▶` / `⏸` 图标（主按钮，高亮显示）
- **下一个视频按钮：** `⏭` 图标
