# PPT 大纲 v3 - 3分钟精简版（含视觉元素）

## 设计原则

- **总页数**：7页（封面 + 5内容页 + 谢谢页）

- **时间分配**：视频/GIF播放时讲解，节省时间

## 设计指南

### 视觉风格

**配色**：
- 背景：深色（#0a0a0a 或类似）
- 主文字：高对比度白色/浅色
- 强调色：品牌色（与 web 一致）
- 辅助色：半透明灰色

**字体**：
- 标题：粗体，大字号
- 正文：中等字号，行距 1.5
- 代码/技术词：等宽字体

**布局**：
- 留白充足
- 视觉焦点明确
- 避免信息过载
- 视频/GIF 占据主要空间

### 时间分配

| 页码 | 内容 | 时间 | 视觉元素 |
|------|------|------|----------|
| 1 | 封面 | 10秒 | 标题 + 背景 |
| 2 | 开场定义 | 20秒 | 工作流图 |
| 3 | PC 演示 | 45秒 | **视频 30-40秒** |
| 4 | VR 演示 | 30秒 | **视频 20-25秒** |
| 5 | 导出价值 | 30秒 | **GIF 动图** |
| 6 | 定位收束 | 35秒 | 图示 + 二维码 |
| 7 | 谢谢页 | 10秒 | 简洁文字 |

**总计**：3分钟

**注意**：视频/GIF 播放时，讲解配合画面进行，节省时间。

### 导出格式

**HTML 版本**（主要）：
- 单文件 HTML
- 内嵌 CSS 和 JS
- 视频/GIF 内嵌或链接
- 可离线运行
- 支持键盘翻页（空格/方向键）
- 适配投影仪分辨率（1920x1080）

**PDF 版本**（备用）：
- 从 HTML 导出
- 视频转为首帧截图 + 播放图标
- 用于 U 盘提交

**图片版本**（备用）：
- 每页导出为 PNG
- 高分辨率（2x）
- 用于紧急情况
