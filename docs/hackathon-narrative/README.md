# 黑客松路演材料 v2 - 酷炫优先版

这组文档用于准备 5 分钟黑客松展示。**叙事逻辑已从”问题-方案”改为”酷炫-价值-技术”**。

## 核心叙事

**产品名**：**一眼成片** (OneGlance)

```text
看一遍，就能发。
极其轻量的 Web 360° 视频剪辑器。
```

**新叙事结构**：
1. **产品名爆点**：一眼成片 - 名字即卖点
2. **这很酷**：VR 演示高光时刻（戴上头显，欣赏一遍，快速成片）
3. **这有价值**：从观看到分享的最短路径
4. **技术实现**：ViewPath 注意力路径
5. **当前完成度**：MVP 闭环 + 试用入口
6. **定位收束**：学生探索 + 影石友好

## 现场策略

```text
主演示：提前录好的 Quest / WebXR 实机视频。
可信度补充：PC 端截图和当前 MVP 闭环。
试用入口：二维码指向现场可访问的真实页面或备用演示页。
备用闭环：即使现场网络或头显失败，也能只靠视频、截图和讲稿完成展示。
```

不要把二维码指向已经删除的 `/hackathon-deck`。二维码应指向真实试用入口，例如现场 LAN 的移动端视频页、XR session 页、PC editor 页，或一个备用公开视频/下载页。
