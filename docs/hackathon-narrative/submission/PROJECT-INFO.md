# 一眼成片 OneGlance

## 项目简介（50字）

一眼成片是极其轻量的 Web XR 360° 视频剪辑器。戴上头显，欣赏一遍视频，系统记录注意力路径，自动渲染成普通视频。看一遍，就能发。

## 核心特点

- 🎯 **一遍过剪辑** - 观看即编辑，无需反复调整
- 💻 **双端支持** - PC 端快捷键 + VR 端头部追踪
- ⚡ **边看边渲染** - 后台自动处理，看完即成片
- 🌐 **Web 轻量** - 无需安装，浏览器直接使用

## 技术架构

- **前端**: Next.js + Three.js + WebXR
- **后端**: Python FastAPI
- **渲染**: FFmpeg 视频处理
- **部署**: Web 应用，支持全平台

## 工作流程

1. 上传 360° 全景视频
2. 进入 VR/PC 编辑器观看
3. 系统记录 ViewPath（注意力路径）
4. 后台自动渲染为 16:9 平面视频
5. 一键下载分享
