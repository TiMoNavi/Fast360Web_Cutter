# 定位

## 一句话

```text
The Invisible Director 是 360 视频的最轻空间导演层：用户在观看 360 素材时按按钮确定观众注意力，系统把这些取景意图变成 ViewPath，并导出成可分享的普通视频。
```

更适合 PPT 标题的短句：

```text
看一遍，就能发。
```

## 为什么不是另一个 360 剪辑器

现场有影石赞助商和评委时，不能把项目讲成“我又做了一个 360 后期工具”。成熟工具已经能做关键帧、重投影、导出和精修，黑客松作品不应该正面挑战这些成熟能力。

更准确的位置是：

```text
成熟工具：完整后期和精修。
The Invisible Director：第一次观看素材时，快速生成取景意图和注意力路径。
```

这不是“替代影石 Studio”，而是探索一个更轻的前置环节：让用户从“我看过素材”更快走到“我可以发出去”。

## 为什么叫空间导演层

360 视频的问题不是只有播放，而是发布前必须回答：

```text
观众这一秒应该看哪里？
这段要不要进成片？
哪里需要切一下？
哪里需要锁住主体？
哪里需要推近？
```

这些判断本质上是导演判断，不是单纯参数编辑。WebXR 的价值是让用户回到 360 素材空间里，用看向、指向和按钮表达判断。

## 对影石友好的表达

固定使用这三句话：

```text
影石这样的相机已经解决了拍到全部。
我探索的是拍完以后，如何更快确定观众注意力。
成熟工具适合精修，我这个原型适合第一次观看时生成取景意图。
```

这三句话能把项目放到生态补充的位置，而不是和赞助商成熟产品竞争。

## 对学生作品友好的表达

不要大谈商业落地，不要说自己已经做成完整生产工具。更适合讲：

```text
这是一个黑客松原型。
它验证的是一种酷炫、直觉、有空间智能味道的工作流。
当前已经跑通上传、session、路径提交、后端 smoke render 和下载。
真实头显采样、快捷按钮和生产级渲染，是下一阶段继续补齐。
```
