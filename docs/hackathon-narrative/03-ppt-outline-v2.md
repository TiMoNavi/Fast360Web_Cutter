## 视觉风格统一要求

1. **配色**：
   - 背景：深色（#0a0a0a 或类似）
   - 主文字：高对比度白色/浅色
   - 强调色：品牌色（与 web 一致）
   - 辅助色：半透明灰色

2. **字体**：
   - 标题：粗体，大字号
   - 正文：中等字号，行距 1.5
   - 代码：等宽字体

3. **布局**：
   - 留白充足
   - 视觉焦点明确
   - 避免信息过载

4. **动画**（HTML 版本）：
   - 页面切换：淡入淡出
   - 元素出现：从下到上滑入
   - 流程图：箭头动画
   - 避免过度动画

---

## 导出格式

1. **HTML 版本**（主要）：
   - 单文件 HTML
   - 内嵌 CSS 和 JS
   - 可离线运行
   - 支持键盘翻页（空格/方向键）
   - 适配投影仪分辨率（1920x1080）

2. **PDF 版本**（备用）：
   - 从 HTML 导出
   - 保留视觉效果
   - 用于 U 盘提交

3. **图片版本**（备用）：
   - 每页导出为 PNG
   - 高分辨率（2x）
   - 用于紧急情况
